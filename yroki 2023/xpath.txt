xPath - язык запросов к элементам xml. Применим к файлам, с xml
структурой.
css - упрощенный вариант. Вниз, право, лево.

Для статического сайта хватит выбрать элемент, пкм, Copy Xpath.
Тег - фиолетовый.
Атрибут - коричневый.
Поиск по xPath начинается с двух слешей.
//тег -> //h3

Уточнение по конкретному значению:
//h3[@название атрибута='значение атрибута']
//h3[@class='header']
взять второй элемент
//h3[3] - взять третий элемент

Уточнение по двум атрибутам
1. //h3[@class='header'][@второй атрибут='его значение']
2. //h3[@class='header' and @второй атрибут='его значение']

Уточнение по одному из значений(содержит)
//h3[contains(@class, 'селектор, содержащий слово')][contains(@class, 'abrakadabra')] - содержит abrakadabra

Уточнение по одному из значений(не содержит)
//h3[contains(@class, 'селектор, содержащий слово')][not(contains(@class, 'abrakadabra'))] - не содержит abrakadabra

Вариант с деревом (внутри div h3)
//div[@id='good']/h3 - один слеш - элемент, под блоком, два слеша - ищем конкретный объект.


ancestor - взять сверху по дереву, отталкивается от текущего элемента.

 //p[@id='iAmLost']/ancestor::h3

Переместится вверх  по дереву от объекта.
Одной точкой - вверх. Двумя точками - на две позиции.

//h3[@id='sister']/..

Найти три элемента одним xPath - or.
//h3[@class='one' or @class='two' or @class='three']

Найти последний элемент
//div[@id=''last]//h3[last()]

Найти предпоследний элемент
//div[@id=''last]//h3[last()-1]

Не равно значению !=
//h3[@price!=5]

Больше/меньше ><
//h3[@price>5]

Больше или равно, меньше или равно
//h3[@courage>=5 and @courage<=15]

Поиск элемента по тексту
//h3[text()='Поиск элемента по тексту']

Поиск элемента, содержащего кусок текста
//h3[contains(text(),'определенное слово')]

Поиск элемента + не содержит
//h3[contains(text(),'определенное слово')][not(contains(text(), 'текст'))]

Поиск элемента без уточнения тега - универсальный элемент
//*


Через css-селекторы.
h3.header.yellow - пробелы заминяются точкой.

css-селектор с названием атрибута
h3.header[thestyle='red'] - по значению.
h3.header[thestyle*='e'] - в середине е.
h3.header[thestyle^='e'] - в начале е.
h3.header[thestyle$='e'] - в конце е.

Спустится на значение вниз в css - #
div#элемент на строку ниже h3

Поиск по атрибуту который не подходит по условию
h3[price]:not([price='0'])
